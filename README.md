# credit-card-transaction-analytics

## Dataset
The dataset contains anonymized credit card transaction records including:
- Customer ID
- Transaction date
- Transaction amount
- Merchant category
- Location information

Source: Kaggle (public dataset)

## Data Model
The raw dataset was logically modeled into analytical tables to improve
scalability, readability, and query performance.

- customers (customer-level attributes)
- transactions (transaction-level facts)
- merchants (merchant categories)
